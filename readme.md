# 🚀 Spawn Engine OS (Firegem)

En avancerad Web3-baserad applikationsplattform byggd med React, Vite och Firebase. 

## 🛠 Tech Stack
- **Framework:** React 18 (TypeScript)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Database/Auth:** Firebase (Firestore & Anonymous Auth)
- **Icons:** Lucide React
